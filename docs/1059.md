# jQuery resize()示例

jQuery **resize()** 事件在浏览器大小改变时被触发，该事件只绑定到 **$(窗口)**。

```
 $(window).resize(function () {
	$('#msg').text('Browser (Width : ' + $(window).width() 
	+ ' , Height :' + $(window).height() + ' )');
}); 
```

要获得浏览器的宽度和高度细节，使用 **$(window)。width()** 和 **$(窗口)。身高()**。
